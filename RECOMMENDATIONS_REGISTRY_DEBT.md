## Known Issue

Some recommendation entries in this registry have placeholder or truncated titles
(e.g., `title: '1'`, `title: '2'`, `title: '--------'`). This occurs because:

1. The underlying research document frontmatter entries were created with
   placeholder titles during initial struct creation
2. Recommendation text extraction from Markdown body did not fully capture
   multi-line entry text
3. Provenance patches attempted to fix these but created partially-applied updates

1. Manually audit docs/research/{doc}.md files with placeholder titles
2. Extract real recommendation summaries from the ## Recommendations Markdown section

5. Commit updated registry
